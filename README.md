# Naman Gupta — Portfolio

Premium, recruiter-focused developer portfolio built with **React + Vite + Tailwind CSS + Framer Motion**.

## ✨ Features
- Dark/light theme toggle, glassmorphism UI, gradient accents
- Hero with typing animation & floating avatar
- Animated counters for achievements
- Interactive timeline of milestones
- Smooth scroll, sticky navbar, scroll progress bar
- Custom cursor glow & subtle hover effects
- Fully responsive (mobile-first)

## 🚀 Getting Started

```bash
npm install
npm start
```

Open http://localhost:3000

## 🏗️ Build

```bash
npm run build
npm run preview
```

## 📁 Structure
```
src/
├── components/      # All UI sections
├── data.js          # Single source of truth (edit me!)
├── App.jsx          # Layout & routing of sections
├── main.jsx         # Entry
└── index.css        # Tailwind + theme
```

## ✏️ Customize
Edit [src/data.js](src/data.js) to update profile, projects, skills, achievements, etc.
Drop your `resume.pdf` inside the `public/` folder to enable the Resume button.
